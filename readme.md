# LazyArch 

1. **Clone the repository:**
```bash
git clone https://github.com/MarceloAntonio/LazyArch

```
2. **Access the project folder:**
```bash
cd LazyArch
```
3. **Make the installer executable.**
```bash
sudo chmod +x Install.sh
```
4. **Run the installer:**
```bash
./Install.sh
```
5. **Run LazyArch anywhere:**
```bash
LazyArch
```
