# Hub Installer Arch 

1. **Clone o repositório:**
```bash
git clone https://github.com/MarceloAntonio/Hub-Installer-Arch

```
2. **Acesse a pasta do projeto:**
```bash
cd Hub-Installer-Arch
```
3. **Instalar dependências:**
```bash
sudo chmod +x Install.sh
```
4. **Execute o main:**
```bash
./hub_arch
```
